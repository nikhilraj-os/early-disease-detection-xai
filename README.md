# 🧠 Early Disease Detection with Explainable AI

## 📌 Overview
This project focuses on predicting diseases at an early stage using machine learning models and explainable AI techniques. It also includes an interactive Power BI dashboard for data visualization and insights.

---

## 🚀 Technologies Used
- Python (Pandas, NumPy)
- Scikit-learn
- Matplotlib
- Power BI

---

## 📊 Features
- Data Cleaning & Preprocessing  
- Exploratory Data Analysis (EDA)  
- Machine Learning Models (Classification)  
- Model Evaluation (Accuracy, Precision, Recall, ROC-AUC)  
- Explainable AI Techniques  
- Interactive Power BI Dashboard  

---

## 📁 Project Files
- `early_disease_prediction.ipynb` → ML model
- `Diabetes_prediction.csv` → Dataset
- `powerbi_dashboard.pbix` → Dashboard file

---

## 📈 Results
- Built predictive models for disease detection  
- Identified key health indicators (Glucose, BMI, etc.)  
- Visualized insights using Power BI
